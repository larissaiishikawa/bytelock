# 🔐 bytelock

A modern password generator web app with a clean hacker-inspired interface.

## ✨ Overview

bytelock is a fast, minimal, and secure password generator designed to help users create strong passwords with ease.
Everything runs locally — nothing is stored or sent externally.

## ⚙️ Features

* Adjustable password length (8–32)
* Toggle character types:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Symbols
* Option to avoid ambiguous characters (O, 0, l, 1)
* Real-time password generation
* Strength indicator (Weak / Medium / Strong)
* Copy to clipboard
* Clean, responsive UI

## 🛠 Tech Stack

* React
* Tailwind CSS

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/bytelock.git
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

## 🔒 Security

* Passwords are generated locally in the browser
* No data is stored or transmitted
* Designed with privacy-first principles
